# A股每日盘前要闻 | A-Stock Daily Pre-Market News

## 简介

本项目计划每日整理并公开分享A股的盘前重要新闻资讯，帮助关心中国资本市场的投资者、分析师、研究员等高效了解每日最新要闻。

- 静态网页，支持 GitHub Pages 托管
- 每天手动或自动更新内容
- 欢迎任何人协作

## 启动方法

1. 通过 https://ysq2003.github.io/A-Stock-Daily-News/ 访问页面。
2. 选择您感兴趣的板块。
3. 点击开始
4. 等待大模型分析
