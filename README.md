# A股每日盘前要闻 | A-Stock Daily Pre-Market News

[English below | 中文见上]

## 简介

本项目计划每日整理并公开分享A股的盘前重要新闻资讯，帮助关心中国资本市场的投资者、分析师、研究员等高效了解每日最新要闻。

- 静态网页，支持 GitHub Pages 托管
- 每天手动或自动更新内容
- 欢迎任何人协作

## 启动方法

1. 将新闻内容写入 `index.html` 文件。
2. 每次更新后，推送至 `main` 分支，GitHub Pages 会自动部署。
3. 可以通过 https://ysq2003.github.io/A-Stock-Daily-News/ 访问页面。

## English

### Introduction

This project aims to organize and share important A-share (China stock) pre-market news daily for investors, analysts, and anyone interested in China’s capital market.

- Static HTML page, published via GitHub Pages
- Manually or automatically updated news
- Collaboration is welcome!

### How to use

1. Write daily news in the `index.html` file.
2. Push updates to `main` branch; GitHub Pages will auto-deploy.
3. Visit https://ysq2003.github.io/A-Stock-Daily-News/ to see the latest news.
